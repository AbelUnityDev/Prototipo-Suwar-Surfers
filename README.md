# Subway Surfers Clon

### Descripción del Proyecto
Este repositorio contiene el proyecto desarrollado para la prueba técnica M5P2. Consiste en un prototipo funcional del género "Endless Runner" similar a Subway Surfers. En este proyecto se implementaron mecánicas de movimiento en carriles, generación procedural de obstáculos, sistema de colisiones complejo y una gestión avanzada de animaciones mediante Blend Trees y Layers.

## **Animaciones**
* Run
* Roll
* DodgeLeft
* DodgeRight
* Jump
* Fall
* Landing
* DethLower
* DeathMovingTrain
* DeathUpper
* DeathBounce
* StumbleLow
* StumbleCornerLeft
* StumbleCornerRight
* StumbleFall
* StumbleOffLeft
* StumbleOffRight
* StumbleSideLeft
* StumbleSideRight
